## Build Setup



### Install
``` bash
# install all dependencies
$ yarn install
```

Copy:
```
cp .env.template .env
```
Configure the file `.env` according to your needs and your local setup.

### Development
``` bash
# serve with hot reload at localhost:3000
$ yarn dev
```

### Build for production
``` bash
# build for production and launch server
$ yarn build
$ yarn start
```
